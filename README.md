# 新的本地写Markdown方法
下载[obsidian](https://obsidian.md/download)，然后git clone本项目，用obsidian打开`source`目录，_posts目录存放着Markdown文章。

### 环境
* Nodejs
* [Pandoc 软件包](https://pandoc.org/installing.html)

```
hexo server -d
```
进入后台 **http://localhost:4000/admin** ，管理页面
